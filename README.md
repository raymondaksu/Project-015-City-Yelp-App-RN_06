<ul>
<li>When the user run the application it is shown that loading indicator center of screen.</li>
<li>User can scroll down and up to see a list of cities and search a city in search bar. Source is pulled API from 'https://opentable.herokuapp.com/api/cities'</li>
<li>When the user tap on the any city, resaurant lists are displayed on the screen with limited details.</li>
<li>User can scroll down and up to see a list of restaurants which belongs to tapped city before. There is a search bar which allow us search a restaurant from the list. Source is pulled API from 'http://opentable.herokuapp.com/api/restaurants'</li>
<li>After tap on a restaurant object, the restaurant details are displayed on the screen.</li>
<li>When clicked on a 'button' of 'Go to Reservation'; redirected to the restaurant page. Shown a tabbar bottom of page which allow us roam in restaurants reservation page, go back, back to apps last page and back to home page.</li>
</ul>
